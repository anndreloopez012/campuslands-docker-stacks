# Docker en macOS

## Pasos

1. Descarga Docker Desktop para Mac segun tu procesador: Apple Silicon o Intel.
2. Arrastra Docker a Applications.
3. Abre Docker Desktop y acepta permisos del sistema.
4. Espera que el motor quede activo.
5. Valida en Terminal:

```bash
docker --version
docker compose version
docker run hello-world
```

## Recomendacion

Si usas Mac con poca memoria, levanta solo los servicios necesarios y apaga los stacks que no estes usando.
