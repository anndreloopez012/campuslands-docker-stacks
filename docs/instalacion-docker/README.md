# Instalacion de Docker

Esta carpeta centraliza las guias para preparar Docker en cada sistema operativo.

## Guias disponibles

- `windows.md`
- `macos.md`
- `linux.md`
- `comandos-basicos.md`

## Validacion rapida

```bash
docker --version
docker compose version
docker run hello-world
```

Si esos comandos funcionan, ya puedes usar cualquier stack del repositorio.
